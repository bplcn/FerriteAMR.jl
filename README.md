An adaptive mesh refinement package written for Ferrite. The package fully originates from the [mk/p4est][p4est] branch of Ferrite toolbox developped by [Maximilian KÃ¶hler][maximilian_koehler]. Necessary modifications are added for version 1.0 of Ferrite.

## Installation
You can install FerriteAMR from the Pkg REPL:
```
pkg> add FerriteAMR
```
